Kodi It Platform

📖 Project Overview

A comprehensive web-based event planning and item rental platform designed to help businesses manage events, inventory, and client bookings efficiently. Built from scratch with scalability in mind.

🎯 Problem & Solution

Problem: Local event planners and rental businesses relied on spreadsheets and manual logs leading to double bookings, lost items, and poor customer experience.

Solution: Developed a centralized system with event management, item inventory, leasing module, and automated invoicing. Provides user roles (admin, vendor, customer) and real-time availability.

⚙️ Features Breakdown

  • Event planning & scheduling
  • Item inventory & rental management
  • Multi-role dashboard (Admin/Vendor/Client)
  • Invoice & PDF generation (DomPDF)
  • Integrated chat support
  • Payment tracking

🛠️ Technologies Used

PHP (OOP) MySQL JavaScript Bootstrap MPDF AJAX HTML5/CSS3

🧩 My Role

Lead backend developer – designed database architecture, implemented core rental logic, built admin panels, and integrated chat system. Also handled UI/UX coordination.

🚀 Challenges & Learnings

Challenge: Creating a real-time availability checker for rental items without conflicts was complex. I used transaction locking and optimized queries to prevent double booking. Additionally, implementing a chat system required learning WebSockets basics; I used polling with AJAX as a fallback.

What I learned: Mastered complex SQL joins and transaction management. Improved problem-solving in inventory systems and learned to balance feature delivery with performance.
